## How to Use This Reference

```js
// Import a component set and create an instance with specific props
const set = await figma.importComponentSetByKeyAsync("COMPONENT_SET_KEY");
const variant = set.children.find(c =>
  c.name.includes("Type=primary") && c.name.includes("size=base")
);
const instance = variant.createInstance();
instance.setProperties({ "Text#22641:241": "Save Changes" });
parent.appendChild(instance);
```

**Key format:** Component properties use `#N:N` suffixes internally (e.g. `Text#22641:241`). When calling `setProperties()`, you must use the full key including suffix as it appears in `componentProperties`. When this reference shows property names, the suffix is stripped for readability — **always check live keys via `instance.componentProperties`.**
